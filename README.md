# Personal-Project
A simple app that tracks sales personnel and records their new sales activities in real time.


# About the project

* Allows field sales executives to enter customer visit details from the app.
* Submits all data in real time for quick and accurate tracking.
* Provides an admin dashboard to view, monitor, and manage all field submissions.


## How I start?

* Built during a free weekend to learn Flutter using a practical, real-world scenario.
* Helped strengthen understanding of Flutter fundamentals and GetX state management.
* Improved basic client communication and requirement-handling skills.
